**“如果当年走了另一条路，你会变成谁？”**

《人生 IF 线 - 高考版》是一款基于纯前端技术（HTML/CSS/Vanilla JS）与大语言模型 (LLM) 驱动的网页互动叙事/文字冒险体验项目。用户可以通过选择不同的年代（如世纪初、一零年代、当下）、心仪的高校以及专业，配合个人性格问卷，生成一段专属的“平行时空大学人生”。

##  核心特性

-  **时代风格沉浸复刻**：针对不同年代设计专属的 UI 界面。例如 `2000.html` 完美复刻了 Windows XP 系统的经典视效以及早期即时通讯软件（类似早期的 QQ/MSN）的聊天界面。
-  **AI 驱动与智能剧情生成**：内置一套灵活的 API 对接系统（在 `config.js` 中配置），能够结合用户的人格画像、学校特色、专业背景，实时流式调用大语言模型生成剧情、选择和人生结局。
-  **海量高校真实设定库**：项目内置了中国数十所顶尖与特色高校的详尽数据，包括且不限于：校园日常、饮食记忆、社团文化、城市链接、隐藏细节以及专属于该校的**“校园梗”**（如北大的“一塔湖图”、哈工大的“规格严格”、厦大的“芙蓉隧道”等），即使在无 AI 离线状态下也能提供极具真实感的随机事件流。
-  **多维人物画像系统**：通过动态力学问卷和互动选择，从社交倾向 (social)、冒险精神 (risk)、理性直觉 (head)、毅力 (grit) 等多个维度对用户进行画像，动态影响学业、社交、恋爱和财富等各项隐藏属性，导向完全不同的毕业结局（如保研、留学、创业、考公、Gap Year 等）。

##  快速开始

本项目为纯静态前端项目，无需配置复杂的构建工具（如 Webpack/Vite）或 Node 环境，开箱即用。

### 1. 基础体验（本地免配置版）

直接在浏览器中双击打开 `index.html`，即可体验完整的 UI 交互与内置的离线故事引擎。程序内置了一套强大的后备文本库，在没有配置 AI 的情况下依然可以完成流畅的叙事闭环。

### 2. AI 沉浸版（需配置 LLM API）

若希望体验更具随机性、文本更丰富的动态剧情，请配置项目中的 `config.js` 文件：

```javascript
// config.js
window.LIFEIF_BUILTIN_API_CONFIG = {
  mockMode: false,      // 关闭 mock 模式以启用真实 API
  streamMode: true,     // 开启流式打字机输出
  jsonMode: true,       // 建议开启，强制模型输出 JSON 格式
  apiUrl: "[https://api.your-llm-provider.com/v1/chat/completions](https://api.your-llm-provider.com/v1/chat/completions)", // 替换为你的大语言模型 API 接口
  apiModel: "your-model-name", // 替换为你使用的模型（如 gpt-4o, claude-3, glm-4 等）
  thinkingType: "disabled", 
  reasoningEffort: "high",
  apiKey: "YOUR_API_KEY_HERE"  // 填入你的真实 API 密钥
};

```

*注：请确保配置的 LLM 模型具有较强的 JSON 格式化输出能力与角色扮演指令遵循能力。*

##  玩法指南

1. **设定时空锚点**：在首页选择你的高考年代（如 2000 年）。
2. **回答内心问卷**：跟随动态浮现的卡片，回答几个关于周末安排、抗压能力、选择偏好等问题，系统将为你生成“AI 画像”。
3. **选择高校与专业**：根据 AI 推荐，或在高校大厅自由挑选一所大学与专业方向，你的选择将直接决定故事的基调与城市背景。
4. **互动故事**：化身 OO（即时通讯软件）用户，与“IF 线小助手”对话。在“大一初到”、“专业分流”、“大二深耕”、“大三规划”、“大四毕业”的每一个节点做出抉择。
5. **保存记忆**：查看最终的十年后写给自己的信、命运轨迹以及系统推荐的书籍和哲理短句，并生成海报。

##  技术栈与实现细节

* **无依赖前端**：为了最大的兼容性与纯粹性，未引入 React/Vue 等框架，全篇使用原生 HTML、CSS 变量、Flex/Grid 布局及 Vanilla JavaScript 编写。
* **动态物理引擎 (Kinetic)**：`index.html` 中的选择卡片采用了自定义的微型物理与视差引擎，跟随鼠标/手指运动产生动态纵深感。
